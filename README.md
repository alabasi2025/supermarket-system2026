# 🏪 نظام إدارة مشتريات السوبرماركت

نظام ويب متكامل لإدارة مشتريات السوبرماركت باللغة العربية.

## 🚀 التقنيات المستخدمة

- **Backend:** Python Flask
- **Database:** SQLite
- **Frontend:** Tailwind CSS + Alpine.js
- **Language:** Arabic RTL

## 📦 المميزات

- ✅ إدارة الموردين
- ✅ إدارة الأصناف والأقسام
- ✅ ربط أسماء المورد بأسمائك الداخلية
- ✅ إعدادات الفرز المقيدة (صنف واحد يتفرز لعدة أصناف)
- ✅ فواتير المورد مع تسجيل الدفعات
- ✅ فاتورة المندوب (الفرز)
- ✅ استلام المخزن
- ✅ تتبع المخزون وحركاته
- ✅ أسعار الموردين والمنافسين
- ✅ تسعيرات البيع المتعددة
- ✅ إدارة المستخدمين والصلاحيات
- ✅ تقارير شاملة

## 👥 أدوار المستخدمين

| الدور | الصلاحيات |
|-------|-----------|
| مدير (manager) | جميع الصلاحيات + إدارة المستخدمين |
| مندوب (agent) | فواتير المورد + الفرز + التسليم |
| أمين مخزن (warehouse) | استلام البضاعة + المخزون |

## 🛠️ التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
pip install flask
```

### 2. تشغيل النظام
```bash
python app.py
```

### 3. فتح المتصفح
```
http://localhost:5000
```

### 4. بيانات الدخول الافتراضية
- **اسم المستخدم:** admin
- **كلمة المرور:** admin

## 💰 العملة
